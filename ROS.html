Virtuoso Robot Operating Systems

Johns Hopkins University

View Website

The Virtuoso Robot is a surgical robot developed by Virtuoso Surgical. The Virtuoso Endoscopic System (VES) delivers two robotically controller, needle-sized manipulators that work from the tip of a rigid endoscope. This system replaces straight tools in almost any rigid endoscopic procedure and enables the surgeon to grasp, manipulate, and cut tissue with dexterity unparalleled in rigid endoscopy today.

The goal of this project was to develop a digital twin of the Virtuoso Robot for simulation, training, and seamless control mode switching between the Surgeon Input Device (SID) and autonomous command inputs.

To achieve this, our team developed a complete ROS 2-based architecture. We modeled the robot’s concentric tube behavior using URDF and integrated it with ROS 2 Control via a custom hardware interface. We implemented control schemes that supported both command-line (CLI) and SID-based control, allowing real-time switching between manual and digital modes. The final system enabled precise tip control in both simulation and hardware, and featured a Gazebo simulation environment to visualize and validate motion prior to physical deployment.
