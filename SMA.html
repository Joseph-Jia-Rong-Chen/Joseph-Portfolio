Smart Memory Alloy Actuator

Purdue University

Smart Memory Alloy (SMA) is a material known for its substantial thermal contraction. This unique property renders it suitable for application as an actuation mechanism. The research centered on characterizing SMA actuation with respect to input current and integrating a control apparatus for employment in a minimally invasive surgical robot designed for lumbar discectomy procedures. Due to the inherently non-linearity behavior, a Gain Scheduling controller was implement to establish successful control.

The experimental setup incorporated a linear bearing to minimize friction, a contactless Hall effect sensor for precise position tracking, and a hanging weight to apply a constant load during actuation. Electrical current was applied to the SMA wire to induce Joule heating, triggering thermal contraction. To ensure responsive and synchronized control, a Real-Time Operating System (RTOS) was implemented on the microcontroller, enabling high-frequency data acquisition and actuation.

The theoretical non-linear characteristic of SMA wire was successfully replicated using custom setup

The system's position response to a step input demonstrated an accuracy of ±0.2 mm and a response time of less than 0.2 s.
